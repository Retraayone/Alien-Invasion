# Alien-Invasion
A space shooter game made using Pygame
# Requirements
- Python
- Code compiler(eg.VsCode)
- Pygame library

# How to use
- download the .zip file from the release.
- Unzip the file and open alien_invasion.py
- run it.


# How to Play
- Left arrow key = move left
- Right arrow key = move right
- Space = Shoot
